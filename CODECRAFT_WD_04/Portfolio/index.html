Project 4

CRUD

The CRUD Operation project is a simple and efficient mobile inventory management system that allows users to manage mobile product details dynamically. Built using HTML, CSS, and JavaScript, this project enables users to add, view, update, and delete mobile inventory records in a structured table format. Users can input details such as Mobile Code, Mobile Name, Quantity, and Price through a form, and upon submission, the data is displayed in a dynamically generated table. JavaScript functions handle CRUD operations, allowing real-time modifications without requiring a database. The project provides a user-friendly interface for managing inventory, making it useful for small-scale product management.

Project 5

CALCULATOR

This Calculator project is a web-based arithmetic calculator built using HTML, CSS, and JavaScript. It provides a user-friendly interface for performing basic mathematical operations such as addition, subtraction, multiplication, and division. The calculator features a numeric keypad and operator buttons, allowing users to input numbers and perform calculations effortlessly. It includes functionalities like AC (Clear), DEL (Delete Last Entry), and Percentage (%) for enhanced usability. JavaScript functions handle the calculations dynamically, using the eval() function to process user inputs.
